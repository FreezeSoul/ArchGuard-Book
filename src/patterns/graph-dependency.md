# 依赖图

维基百科：https://en.wikipedia.org/wiki/Dependency_graph

在数学、计算机科学和数字电子学中，依赖图是表示多个对象相互依赖的有向图。 可以从依赖图中导出评估顺序或不存在尊重给定依赖关系的评估顺序。
