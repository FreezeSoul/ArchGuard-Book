# 软件架构：分析、治理的模式与实践

对于架构分析、治理的相关总结：

- 第一部分，基础篇，介绍了架构治理的基本概念，包括架构治理的目标、架构治理的方法等。
- 第二部分，模式篇，介绍架构治理的相关模式。
- 第三部分，度量篇，介绍了架构治理的度量方法。
- 第四部分，工具篇，介绍了相关的架构治理工具。
